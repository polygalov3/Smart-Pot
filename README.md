# Smart-Pot
Smart Pot
Автоматическая система полива растения для поддержания оптимального значения влажности и Ph

Задача: Спроектировать и сконструировать систему, информация которой будет отображаться на сайте, полива вазона
Реализация: Arduino считывает влажность, pH и уровни баков, и через Modbus RTU управляет двумя перистальтическими насосами для подачи воды и раствора. ESP8266 передаёт текущие значения по Wi-Fi, обеспечивая полностью автономную работу системы.
